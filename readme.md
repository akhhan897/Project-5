## Sprint Goal

Build a functional Photo Sharing Application that:
- Displays users and their details
- Allows navigation between users and photos
- Displays photos and comments
- Fetches data from the backend using FetchModel
- Updates TopBar dynamically based on context

---

## Scrum Process

This project follows Scrum methodology using a GitHub Project Board with the following columns:
- To Do
- In Progress
- Code Review
- Testing
- Done

Issues were created as Agile user stories and tracked through each stage of development.
